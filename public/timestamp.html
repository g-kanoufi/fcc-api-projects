Timestamp microservice

This microservice, once passed a string as a parameter, will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)

If it does, it returns both the Unix timestamp and the natural language form of that date else it returns null for those properties.

Example usage:

  • https://fcc-api-projects-g-kanoufi.c9users.io/timestamp/December%2015,%202015
  • https://fcc-api-projects-g-kanoufi.c9users.io/timestamp/1450137600

Example output: 

{ 
	unix: 1450137600, 
	natural: "December 15, 2015" 
}

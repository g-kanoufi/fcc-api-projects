Url shrtnr microservice

This microservice, once passed a url as a parameter, will check to see whether that url is a real url format (example: http:www.freecodecamp.com)

If it does, it returns both original and shortened url else it returns an error.

Example usage:

  • https://fcc-api-projects-g-kanoufi.c9users.io/shrt/new/http://www.freecodecamp.com
  • https://fcc-api-projects-g-kanoufi.c9users.io/shrt/new/http://codepen.io/collection/nLPwkP/

Example output: 

{ 
	original: "http://codepen.io/collection/nLPwkP/", 
	shortened:  "https://fcc-api-projects-g-kanoufi.c9users.io/shrt/cdp"
}
